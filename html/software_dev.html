Duties of a Computer Research Scientist!

  • Develop and improve Software Systems.
  • Create experiments to test software operation.
  • Solve problems and develop theories in the fundamental level of computing.
  • Work together with scientists and engineers to improve computers.
  • Invent programming languages and tools that improve developer and user interaction with computing in general.

Computer Scientists work to both simplify and increase the efficiency of modern computing algorithims, paving the way for modern technology like machine learning and quantum computing.

Computer Scientists normally specialize in Robotics, Data Mining, or Programming.
